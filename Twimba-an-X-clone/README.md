## 📂 How to Run Locally

```bash
# Clone the repository
git clone [https://delta-r-git.github.io/scrimba-projects/Twimba-an-X-clone.git](https://delta-r-git.github.io/scrimba-projects/Twimba-an-X-clone.git)

# Navigate to the project directory
cd scrimba-projects/Twimba-an-X-clone

# Open index.html in your default browser
open index.html
```
